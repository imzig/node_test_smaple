## express 설치

```
$ npm install express --save
```

## MongoDB 설치
```
$ brew install mongodb
$ mkdir -p /data/db
$ sudo chown {userid} /data/db
```
### MongoDB 데몬 실행
```
$ mongod
```
### MongoDB 접속
```
$ mongo
```

## Mysql 설치
```
$ brew install mysql
```
### Mysql 데몬 실행
```
$ mysql.server start
```
